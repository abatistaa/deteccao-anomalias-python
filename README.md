# Detecção de Anomalias em Transações Financeiras com Python - Bootcamp Accenture

## 🎯 Objetivo do Projeto
Este projeto foi desenvolvido como parte do Bootcamp Accenture/DIO, com o foco em aplicar lógica de programação e análise de dados para identificar padrões suspeitos em transações financeiras. A detecção de anomalias é uma peça fundamental na prevenção de fraudes e na garantia da segurança de dados bancários.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Python 3.x
* **Bibliotecas Sugeridas:** Pandas (Manipulação de dados), Matplotlib/Seaborn (Visualização de Outliers).
* **Ambiente:** Google Colab / Jupyter Notebook.

## 🧠 Conceitos Aplicados
Durante o desenvolvimento, foram explorados conceitos essenciais de análise de dados voltada à segurança:
* **Identificação de Outliers:** Localização de transações que divergem drasticamente da média de consumo do usuário.
* **Análise de Falsos Positivos:** Refinamento da lógica para evitar que compras legítimas sejam bloqueadas indevidamente.
* **Limiares de Decisão (Thresholds):** Definição de valores críticos para disparo de alertas de segurança.

## 🚀 Como Executar
1. Clone o repositório.
2. Certifique-se de ter o Python instalado.
3. Execute o script de análise (ex: `detector_fraude.py`) para visualizar as transações sinalizadas.
